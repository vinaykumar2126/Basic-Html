Learning HTML for beginners

Defination

HTML stands for Hyper Text Markup Language.It is used to design and develop Web Pages. Tim Berners-Lee invented the HTML.The code used to make them visually appealing is known as CSS and we shall focus on this in a later tutorial.HTML provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items.
